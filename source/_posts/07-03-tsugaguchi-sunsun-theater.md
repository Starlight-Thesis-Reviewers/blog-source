# 1. 什么是塚口 SUNSUN 剧场

塚口 SUNSUN 剧场（以下简称 SUNSUN）是一家位于兵库县尼崎市的电影院，今年（2022 年）迎来了创立 69 周年^[[“于昭和 28 年（1953 年）12 月 28 日，以‘塚口剧场’之名开馆”](https://sun3tabloid.wordpress.com/2020/04/09/%e3%81%af%e3%81%98%e3%82%81%e3%81%be%e3%81%97%e3%81%9f%e3%80%82/)]。尽管外观上只是一座普通的“街边电影院”，但为何在我心中留下了独一无二的 Revue 体验呢？本文首先将介绍 SUNSUN 是一座怎样的电影院，然后探讨“塚口的 Revue”的魅力所在。

SUNSUN 剧场首要且不可或缺的特色，当属因其卓越音效而吸引全国影迷前来体验的“特别音响放映”。SUNSUN 对音效的执着始于《疯狂的麦克斯：狂暴之路》（2015 年 6 月上映）^[2015 年上映。在本作结尾部分，华恋和光在 Revue 中，华恋化为十字架并冲入沙漠风暴的场景，其灵感来源正是这部作品。]，该片亦与影院渊源颇深。这可谓是一种承诺吧？

“不对哦，这是命运——”

话说回来，真正决定 SUNSUN 音响方向性的，是 2015 年 11 月上映的《少女与战车 剧场版》。在本片音响监督岩浪美和^[曾在《少女与战车》、《刀剑神域》等多部动画作品中担任音响监督，除了 SUNSUN 剧场外，还亲自前往立川 Cinema City、川崎 Cinecitta 等全国多家影院进行音响调试的一位奇人。]的建议下，影院更换了全套扬声器，装备了堪称“城郊影院”（岩浪命名）的超规格音响设备。不仅如此，他们并非只是配齐设备了事，还会对每一部作品的音效进行试映并细致调整。遗憾的是，在现代的多厅影院，放映技师常常缺席，许多影片不经试映便直接上映。但 SUNSUN 却拥有技术娴熟的放映技师坐镇，这是一家怀着“希望观众能体验到唯有在电影院才能享受到的时光”这一信念来呈现作品的影院。

# 2. 塚口的 Revue（设备·装饰篇）

SUNSUN 的幕布是宽银幕尺寸（CinemaScope size）^[宽高比为1:2.35。相比电视屏幕，其画面更显宽。]（以下简称宽幅）。在放映如《剧场版 少女☆歌剧 Revue Starlight》（以下简称“本作”）这类使用宽幅制作的作品时，预告片以标准尺寸（Vista size）^[宽高比为1:1.85。大致相当于电视屏幕的尺寸（严格来说略有不同）。]放映，而在正片放映前，左右两侧的黑色幕布（即可变遮幅）会打开，呈现出更为广大的宽幅幕布。如今主流的大型影院大多已不设这种可变遮幅，因此，像 SUNSUN 这样能以宛如舞台大幕拉开般的仪式感开启观影的影院实属罕见。从电影放映前那一刻起“我们已经站在舞台之前”。

此外，在 SUNSUN 中，还存在着由志愿者员工组成的手工艺部^[《打造受人喜爱的电影院：塚口 SUNSUN 剧场的轨迹与奇迹》 - [第 11 章 员工奋战篇 “尽我职责！” 4/4](http://www.cinepre.biz/archives/29109)]，迄今为止休息区内已经刷新过坦克和城堡。而在本作中，长颈鹿、番茄以及 Position 0 也都出现在了休息区。长颈鹿的鬃毛等细节均得到了精细还原，其完成度之高，即便听到津田健次郎用配音说出“我懂的”也毫不令人意外。能从 Position 0 的位置眺望长颈鹿，再移步至影院观影——这样的电影院可谓罕见。

# 3. 塚口的 Revue（音响篇）

SUNSUN 特别音响放映的特性，首先在于其对本作这类音乐电影所展现的丰沛音场构建能力。虽然银幕前的低音炮体量巨大，容易凸显重低音的强劲，但高音域与中音域也同样清晰透彻，不挑上映作品的类型。这一点在开场后便能立刻感受到。电影配乐《color temperature》的引子部分，剧场内回荡着乐队的强音，仿佛让人瞬间误入一场音乐会。而《世界是我们的…》（世界は私たちの…）从 2 分 53 秒起的铜管乐段，以及《wi(l)d-screen baroque》中的弦乐旋律，其舒展与狂野的鸣响，是 CD 音源根本无法再现的。

SUNSUN 特别音响放映的另一大魅力在于台词易于听清。尽管通过强化音乐和音效可以极大提升影片的冲击力与临场感，但 SUNSUN 所秉持的理念是：构成电影根基的，始终是台词。

环绕声的出色表现亦是一大看点。例如，在地铁场景中（作为舞台装置），血雨倾盆而下，真矢大人厉声喝斥“别被吓着了！”的片段；或是在《MEDAL SUZDAL PANIC ◎〇●》中，真昼不断重复“就是你就是你就是你——”的片段。前者让人仿佛亲身淋到血雨般惊慌失措，并对真矢大人的喝斥感到由衷震撼；后者则令人产生正被真昼四处追逐的错觉，体会到真实的恐惧。这固然得益于精良的音响调校，同时，由于该剧院的规模相较于大型连锁影院更为小巧，从扬声器直接传入耳中的“直接音”在声音中所占比例更高，而非经由剧场墙壁与天花板反射而来的“间接音”，这一点也起到了加分作用。

# 4. 推荐塚口的 Revue

这就是塚口 SUNSUN 剧场——乍一看只是家寻常的街边电影院，实则是一家洋溢着对电影的热爱、独一无二的影院。另外，本文虽未详述，SUNSUN 剧场还有“活动放映”^[《打造受人喜爱的电影院：塚口三三剧场的历程与奇迹》 - [第6章 前言 艺人篇 “只有灵魂可以燃烧！” 3/4](https://www.cinepre.biz/archives/28572)]这一特色，即便在新冠疫情期间也推行了诸如“无发声应援放映”等活动。鉴于本部作品也包含了适合应援放映的看点，例如存在观众可以对其中的歌舞桥段报以掌声的段落，我们期待未来能看到更多由 SUNSUN 剧场举办的放映活动——这正是因为它已在众多活动放映中积累了丰富经验，方能呈现的特色。

笔者希望本作能够再次上映，使其成为一处能从多种角度体验作品原汁原味魅力的场所。
